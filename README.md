# accessibility-exercises

A collection of small exercises used in the She Codes Accessibility lesson.

Each folder focuses on a different accessibilty concept and includes starter files for the exercise and a solution folder with one possible implementation.

Exercises

percent-layout
Working with percentage units to create flexible layouts.

final-exercise-responsive-page
A larger exercise that combines multiple responsive techniques.

These exercises are intended to be explored during the lesson and practiced in the browser.
